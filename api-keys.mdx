![generate API key](https://res.cloudinary.com/curead/image/upload/v1717754210/Shootmail/mocks/Screenshot_2024-06-07_at_3.24.59_PM_wnatbk.png)

After you log in into your Shootmail account, click on your profile picture on the top right and go to [API Keys](https://shootmail.app/settings/api-keys).

Please note, new API keys generated will only be shown once, please copy the key and keep it safe. You cannot retrieve it later
but you can always generate a new one.
